# BlockShell
A drag-and-drop, Scratch-style CLI builder. Build terminal commands like LEGO blocks.

## Features
- Visual CLI command builder
- Real-time bash preview
- Built-in terminal emulator

# Clone and Run
    

### 1. Clone the repo

### 2. Install dependencies
npm install

### 3. Start the app
